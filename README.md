# Mini IT Ticketsystem

## Kurzbeschreibung
Dieses Projekt ist ein Mini IT Ticketsystem, das ich als persönliches Projekt erstellt habe. Es simuliert ein einfaches Support-System, in dem Tickets von Benutzern erstellt, angezeigt und als erledigt markiert werden können.  

Es dient dazu, meine Fähigkeiten in HTML, CSS und JavaScript zu demonstrieren und zeigt mein Interesse an IT-Support und Problemlösungen.

## Funktionen
- Neues Ticket erstellen (Name + Problembeschreibung)  
- Tickets werden in der Tabelle angezeigt  
- Status jedes Tickets: Offen / Erledigt  
- Tickets bleiben gespeichert im Browser (localStorage)  
- Button zum Markieren eines Tickets als erledigt  

## Verwendete Technologien
- HTML5  
- CSS3  
- JavaScript (DOM-Manipulation, localStorage)  

## Nutzung
1. Repository herunterladen oder GitHub-Seite öffnen  
2. `index.html` im Browser öffnen  
3. Ticketformular ausfüllen und absenden  
4. Tickets in der Tabelle anzeigen lassen und erledigte Tickets markieren  

## Was ich gelernt habe
- Erstellung eines kleinen Webprojekts von Grund auf  
- Arbeit mit Formularen und DOM-Manipulation in JavaScript  
- Speicherung von Daten im Browser mit localStorage  
- Umsetzung eines Projekts, das typische IT-Support-Prozesse simuliert
